About This Project

Rising Waters — A Machine Learning Approach to Flood Prediction

Project Overview

Rising Waters is an intelligent flood risk prediction system that uses machine learning to analyse meteorological and environmental data, helping disaster management authorities make timely, data-driven decisions.

The system trains and compares four classification models — Decision Tree, Random Forest, K-Nearest Neighbors, and XGBoost — selecting the best performer (XGBoost, ~90.80% accuracy) for real-time deployment via a Flask web application.

Tech Stack
Python 3.x NumPy Pandas Scikit-Learn XGBoost Matplotlib Seaborn Flask Pickle Joblib SQLite IBM Cloud
ML Pipeline
  1. Import Libraries — NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, XGBoost, Pickle
  2. Read Dataset — pd.read_excel / pd.read_csv with head(), shape, describe()
  3. Exploratory Data Analysis — correlation heatmap, target distribution
  4. Univariate Analysis — distribution plots (histplot) & box plots per feature
  5. Handle Missing Values — median imputation to fill null entries
  6. Handle Outliers — IQR-based capping (preserves dataset size)
  7. Handle Categorical Values — feature mapping & label encoding
  8. Data Preprocessing — train/test split (80/20), StandardScaler
  9. Model Training — Decision Tree, Random Forest, KNN, XGBoost
  10. Evaluation — confusion matrix, classification report, 5-fold cross-validation
  11. Save Best Model — XGBoost saved as floods.save using Pickle
